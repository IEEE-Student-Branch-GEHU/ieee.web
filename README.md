### Objectives

- Facilitate networking with industry professionals and IEEE members worldwide.
- Promote research and development activities among students.
- Contribute to society through technical outreach programs.

## 🛠️ Technical Stack
This website is built using modern web technologies:
- **Frontend**: React (Vite), Tailwind CSS v4
- **Backend**: Node.js, Express
- **Reporting**: Integrated with IEEE vTools

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1.  **Clone the repository**
    ```bash
    git clone https://github.com/YZO-BIT/IEEE-website.git
    cd IEEE-website
    ```

2.  **Setup Backend**
    ```bash
    cd backend
    npm install
    npm start
    ```

3.  **Setup Frontend**
    ```bash
    cd frontend
    npm install
    npm run dev
    ```
