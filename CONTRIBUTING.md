# Contributing to IEEE GEHU Website 🌐

Welcome! We're thrilled that you're interested in contributing to the official repository.

## 📜 Code of Conduct
Please be respectful and professional in all interactions.

## 🚀 How to Contribute

1. **Find an Issue**: Look for open issues or open a new one to discuss your ideas.
2. **Fork & Branch**: Fork the repo and create a branch for your feature (`feat/your-feature`) or fix (`fix/your-fix`).
3. **Write Code**: Follow the existing coding style and patterns.
4. **Test Your Changes**:
   - Frontend: `npm test`
   - Backend: `npm test`
5. **Submit a PR**: Provide a clear description of your changes using our PR template.

## 🛠️ Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS v4, Framer Motion.
- **Backend**: Node.js, Express, MongoDB Atlas, Cloudinary.

## ✅ Pull Request Requirements
- Code must pass all automated CI checks.
- Changes should be documented in the PR description.
- High-quality UI additions should include a screenshot/recording.
